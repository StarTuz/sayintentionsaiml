# ATC Phraseology Guide

> **30 Canonical VFR Exchanges for Stratus ATC**
> Each exchange shows exact pilot/ATC phraseology

---

## Flight Following

### 1. Initial Contact

```
PILOT: "NorCal Approach, Cessna 12345, VFR request"
ATC:   "Cessna 12345, NorCal Approach, go ahead"
```

### 2. Flight Following Request

```
PILOT: "Cessna 12345 is a Cessna 172, 10 miles south of Sacramento, 4500, request flight following to Oakland"
ATC:   "Cessna 12345, squawk 4521"
```

### 3. Radar Contact

```
ATC:   "Cessna 12345, radar contact, 10 miles south of Sacramento. Altimeter 29.92"
PILOT: "29.92, Cessna 12345"
```

### 4. Traffic Advisory - Standard

```
ATC:   "Cessna 12345, traffic 2 o'clock, 3 miles, southbound, altitude indicates 5500"
PILOT: "Traffic in sight, Cessna 12345"
```

### 5. Traffic Advisory - Negative Contact

```
ATC:   "Cessna 12345, traffic 11 o'clock, 5 miles, opposite direction, type unknown"
PILOT: "Negative contact, looking, Cessna 12345"
```

### 6. Altitude Change Request

```
PILOT: "Cessna 12345, request climb to 6500"
ATC:   "Cessna 12345, climb at pilot's discretion, traffic no factor"
```

### 7. Frequency Change

```
ATC:   "Cessna 12345, contact Oakland Approach on 127.0"
PILOT: "127.0, Cessna 12345, good day"
```

### 8. Cancel Flight Following

```
PILOT: "Cessna 12345, field in sight, cancel flight following"
ATC:   "Cessna 12345, radar service terminated, squawk VFR, frequency change approved"
```

## Pattern Operations (Class D)

### 9. Inbound Call

```
PILOT: "Sacramento Tower, Cessna 12345, 10 miles north, inbound for landing with information Alpha"
ATC:   "Cessna 12345, Sacramento Tower, enter left downwind runway 20, report midfield"
```

### 10. Pattern Entry Acknowledgment

```
PILOT: "Left downwind 20, will report midfield, Cessna 12345"
```

### 11. Position Report (Midfield)

```
PILOT: "Cessna 12345, midfield left downwind"
ATC:   "Cessna 12345, number 2, follow Piper on base"
```

### 12. Traffic in Sight

```
PILOT: "Traffic in sight, number 2, Cessna 12345"
```

### 13. Base Turn

```
PILOT: "Cessna 12345, turning left base"
ATC:   "Cessna 12345, continue, cleared to land runway 20"
```

### 14. Landing Clearance Readback

```
PILOT: "Cleared to land runway 20, Cessna 12345"
```

### 15. Cleared for the Option

```
ATC:   "Cessna 12345, cleared for the option runway 20, wind 180 at 10"
PILOT: "Cleared for the option runway 20, Cessna 12345"
```

### 16. Go Around

```
ATC:   "Cessna 12345, go around, traffic on the runway"
PILOT: "Going around, Cessna 12345"
```

### 17. Touch and Go

```
PILOT: "Cessna 12345, touch and go"
ATC:   "Cessna 12345, make left closed traffic"
```

## Ground Operations

### 18. Radio Check

```
PILOT: "Sacramento Ground, Cessna 12345, radio check"
ATC:   "Cessna 12345, loud and clear"
```

### 19. ATIS Request

```
PILOT: "Sacramento Ground, Cessna 12345 at Signature, VFR to Oakland, request taxi with Alpha"
```

### 20. Taxi Clearance (Simple)

```
ATC:   "Cessna 12345, runway 20, taxi via Alpha"
PILOT: "Runway 20 via Alpha, Cessna 12345"
```

### 21. Taxi Clearance (Hold Short)

```
ATC:   "Cessna 12345, taxi to runway 20 via Alpha, hold short of runway 16"
PILOT: "Taxi to runway 20 via Alpha, hold short runway 16, Cessna 12345"
```

### 22. Run-Up Request

```
PILOT: "Cessna 12345, ready for run-up"
ATC:   "Cessna 12345, run-up approved, advise ready"
```

### 23. Ready for Departure

```
PILOT: "Cessna 12345, ready for departure runway 20"
ATC:   "Cessna 12345, hold short runway 20, traffic on final"
```

### 24. Takeoff Clearance

```
ATC:   "Cessna 12345, runway 20, cleared for takeoff"
PILOT: "Cleared for takeoff runway 20, Cessna 12345"
```

### 25. Departure Instructions

```
ATC:   "Cessna 12345, fly runway heading, contact departure"
PILOT: "Runway heading, Cessna 12345"
```

## Quick Reference: Response Types

| Instruction | Readback Required | Acceptable Response |
|:------------|:------------------|:--------------------|
| Squawk code | ✅ Yes | "Squawk 4521" |
| Hold short | ✅ Yes | Full readback |
| Runway assignment | ✅ Yes | Include runway number |
| Frequency | ✅ Yes | Include frequency |
| Traffic advisory | ❌ No | "Traffic in sight" or "Negative contact" |
| Altimeter | IFR: Yes, VFR: No | Read back setting |
| Radar contact | ❌ No | "Roger" or acknowledge |

## Prohibited Phraseology

❌ **Never say**:

- "Cleared to takeoff" (on ground) - only Tower can clear
- "Taking off" - use "departing" instead
- "With you" - just state position
- "Back to you" - use proper handoff
- "Affirmative" for clearances - use specific readback

✅ **Always use**:

- "Wilco" - Will comply
- "Roger" - Message received
- "Unable" - Cannot comply
- "Say again" - Request repeat
